## Getting Started ğŸ

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm package manager
- Angular CLI v16.2.0

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/ahmedkhaleddev02-del/Fresh-Cart-Angular.git
   ```

2. Navigate to the project directory

   ```bash
   cd Fresh-Cart-Angular
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the development server

   ```bash
   ng serve
   ```

5. Open your browser and navigate to `http://localhost:4200`
